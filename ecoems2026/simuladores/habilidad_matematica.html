Sucesiones Numéricas

Identificar patrones en secuencias de números y determinar términos siguientes o faltantes.

1.1 Sucesiones Aritméticas

Definición: Secuencia donde la diferencia entre términos consecutivos es constante.

Fórmula: aₙ = a₁ + (n-1)d

Ejemplo: 2, 5, 8, 11, 14, ...

Diferencia: +3. Siguiente término: 17

1.2 Sucesiones Geométricas

Definición: Secuencia donde la razón entre términos consecutivos es constante.

Fórmula: aₙ = a₁ × rⁿ⁻¹

Ejemplo: 3, 6, 12, 24, 48, ...

Razón: ×2. Siguiente término: 96

1.3 Sucesiones Cuadráticas

Definición: Secuencia donde la segunda diferencia entre términos es constante.

Fórmula: aₙ = an² + bn + c

Ejemplo: 1, 4, 9, 16, 25, ...

Patrón: n². Siguiente término: 36

1.4 Sucesiones Alternadas

Definición: Secuencia con dos patrones diferentes que se alternan.

Ejemplo: 2, 4, 3, 6, 5, 10, 9, ...

Patrón: ×2, -1. Siguiente término: 18

1.5 Sucesión de Fibonacci

Definición: Cada término es la suma de los dos anteriores.

Ejemplo: 1, 1, 2, 3, 5, 8, 13, ...

Siguiente término: 21 (8 + 13)

1.6 Estrategias para Identificar Patrones

  • Calcular diferencias entre términos consecutivos
  • Buscar multiplicadores o divisores constantes
  • Identificar patrones de posición (n², n³, etc.)
  • Observar si hay dos patrones alternados
  • Verificar si es una sucesión famosa (Fibonacci, primos, etc.)

Problemas de Razonamiento

Resolver situaciones lógicas con números mediante deducción y eliminación.

4.1 Problemas de Relaciones y Condiciones

Definición: Establecer relaciones entre elementos basadas en condiciones dadas.

Ejemplo: "Ana es más alta que Beatriz. Carmen es más baja que Beatriz. ¿Quién es la más alta?"

Solución: Ana > Beatriz > Carmen → Ana es la más alta

4.2 Problemas de Edad

Definición: Determinar edades actuales o futuras basadas en relaciones temporales.

Ejemplo: "La edad de Juan es el doble que la de Pedro. Hace 5 años, la suma de sus edades era 40. ¿Cuántos años tiene Juan?"

Solución: J = 2P, (J-5)+(P-5)=40 → 2P-5+P-5=40 → 3P=50 → P=16.67, J=33.33

4.3 Problemas de Trabajo

Definición: Calcular tiempos de trabajo individuales y conjuntos.

Ejemplo: "A puede hacer un trabajo en 6 horas, B en 4 horas. ¿En cuánto tiempo lo harán juntos?"

Solución: 1/6 + 1/4 = 1/t → 2/12 + 3/12 = 5/12 = 1/t → t = 12/5 = 2.4 horas

4.4 Problemas de Velocidad

Definición: Calcular distancias, tiempos o velocidades.

Ejemplo: "Un auto viaja a 60 km/h. ¿Cuánto tiempo tarda en recorrer 120 km?"

Solución: t = d/v = 120/60 = 2 horas

4.5 Estrategias para Resolver Problemas de Razonamiento

  • Leer cuidadosamente el problema
  • Identificar la información clave
  • Establecer variables y ecuaciones
  • Utilizar diagramas o tablas cuando sea útil
  • Verificar que la solución satisface todas las condiciones
